# Meteostation
Необходимо:
---
* Arduino nano v3
* Барометр BMP180
* OLED экран 128*128
---
* Библеотека Wire
* Библеотека Adafruit_GFX
* Библеотека Adafruit_SSD1306
---
Установка:
1. Архивы распаковать в /user/Documents/Arduino/libraries
2. Код загрузить на плату
---
Рекомендации:
* Выпаять/Обрезать светодиоды на плате (L и ON)
